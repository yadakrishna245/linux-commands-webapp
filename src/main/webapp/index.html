🐧 Linux 100 Commands Cheat Sheet

📁 File & Directory

  • ls - list files
  • ls -la - detailed list
  • cd - change directory
  • pwd - current path
  • mkdir - create directory
  • rmdir - remove empty dir
  • rm - delete file
  • rm -rf - force delete dir
  • cp - copy files
  • mv - move/rename
  • touch - create file
  • stat - file info
  • tree - directory tree
  • basename - extract filename
  • dirname - extract directory

📖 File Viewing

  • cat - show file content
  • less - view file scroll
  • more - paginated view
  • head - first lines
  • tail - last lines
  • tail -f - live logs
  • nl - show line numbers
  • wc - word count
  • file - file type
  • strings - readable content

🔍 Search & Text

  • grep - search text
  • grep -r - recursive search
  • find - find files
  • locate - fast search
  • which - command path
  • whereis - binary location
  • awk - text processing
  • sed - stream edit
  • cut - extract columns
  • sort - sort lines
  • uniq - remove duplicates
  • tr - replace chars

⚙️ System

  • top - processes
  • htop - advanced view
  • uptime - system uptime
  • free -m - memory
  • df -h - disk usage
  • du -sh - folder size
  • uname -a - system info
  • hostname - system name
  • date - current date
  • cal - calendar

🌐 Networking

  • ping - connectivity
  • ifconfig - network config
  • ip a - IP info
  • netstat - network stats
  • ss - socket stats
  • curl - fetch URL
  • wget - download
  • scp - copy remote
  • ssh - remote login
  • ftp - file transfer

👤 Users

  • whoami - current user
  • who - logged users
  • id - user info
  • adduser - add user
  • userdel - delete user
  • passwd - change password
  • su - switch user
  • sudo - root command

🔐 Permissions

  • chmod - change permission
  • chown - change owner
  • chgrp - change group
  • umask - default perms

📦 Packages

  • apt update - update
  • apt upgrade - upgrade
  • apt install - install
  • apt remove - remove
  • yum install - install RHEL
  • dnf install - modern install

🧠 Process

  • ps aux - process list
  • kill - stop process
  • kill -9 - force kill
  • bg - background
  • fg - foreground
  • jobs - job list
  • nohup - run persistent

📂 Compression

  • tar -cvf - create tar
  • tar -xvf - extract tar
  • gzip - compress
  • gunzip - unzip
  • zip - zip file
  • unzip - unzip file

🛠️ Disk

  • mount - mount disk
  • umount - unmount
  • lsblk - block devices
  • fdisk - partition
  • mkfs - format disk
  • blkid - disk UUID

📜 Misc

  • history - command history
  • clear - clear screen
  • echo - print text
  • env - environment vars
  • export - set variable
  • alias - shortcut cmd
  • unalias - remove alias
