🐧 Linux Commands Cheat Sheet

📁 File Commands

  • ls - list files
  • ls -la - detailed list
  • cd - change directory
  • pwd - current directory
  • touch file - create file
  • cp src dest - copy file
  • mv src dest - move/rename
  • rm file - delete file
  • rm -rf dir - delete directory

⚙️ System Commands

  • top - live processes
  • htop - advanced process view
  • df -h - disk usage
  • du -sh - folder size
  • free -m - memory usage
  • uptime - system uptime
  • whoami - current user

🌐 Networking

  • ping google.com - check connectivity
  • ifconfig - network info
  • ip a - IP details
  • netstat -tulnp - ports
  • ss -tulnp - socket stats
  • curl url - fetch data
  • wget url - download file

👤 User Commands

  • adduser user - create user
  • passwd user - set password
  • su user - switch user
  • who - logged users
  • id - user info

🔐 Permissions

  • chmod 755 file - change permission
  • chown user:file - change owner
  • umask - default permissions

📦 Package Commands

  • apt update - update packages
  • apt upgrade - upgrade system
  • apt install pkg - install package
  • yum install pkg - install (RHEL)
  • dnf install pkg - modern RHEL

🧠 Process Commands

  • ps aux - process list
  • kill PID - kill process
  • kill -9 PID - force kill
  • bg - background process
  • fg - foreground process
